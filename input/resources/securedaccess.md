# SecuredAccess

<section>

## Overview

A generic resource for exposing a workload by creating the
necessary service and ingress resources and optionally
generating TLS credentials.


</section>

<section>

## Spec properties

- **ports** _array_

- **selector** _object_

- **ca** _string_

- **certificate** _string_

- **accessType** _string_

- **options** _object_

</section>
