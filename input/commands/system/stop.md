# System stop command

```shell
skupper system stop [options]
```

Stop the Skupper router for the current site.  This stops the
systemd service for the current namespace.

Platforms: Kubernetes, Docker, Podman, Linux

## Primary options

## Global options

--namespace
Type: (-n) <name>
Flags: global

Set the current namespace.
