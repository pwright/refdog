# skupper site delete

Delete a site.


## Usage

~~~ shell
$ skupper site delete <name>
Waiting for deletion to complete...
Site "<name>" is deleted
~~~

## Arguments

- **name** _string_

  The name of the site resource.
  

## Errors

- **No site resource exists**

  There is no existing Skupper site resource to delete.
