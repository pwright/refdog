# skupper connector delete

<section>

Delete a connector.

</section>

<section>

## Usage

~~~ shell
$ skupper connector delete <name>
Waiting for deletion to complete...
Connector "<name>" is deleted
~~~

</section>

<section>

## Arguments

- <h3 id="name">name <span class="argument-info">string</span></h3>

  The name of the connector resource.

</section>
