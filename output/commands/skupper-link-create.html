skupper link create

Create a link.

A link is a site-to-site communication channel. Links serve as a transport for application connections and requests.

Usage

$ skupper link create <file> [options]
Waiting for status...
Link "<name>" is active
You can now safely delete <file>

Arguments

  • file None

    The name of the token file.

  • --cost integer
